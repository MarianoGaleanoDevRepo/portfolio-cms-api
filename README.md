
# 🚀 Portfolio CMS API

💡 Plataforma fullstack para gestionar y mostrar proyectos profesionales desde un panel moderno.

🎯 Objetivo

Crear un sistema donde puedas:

🧑‍💻 Administrar tu portafolio
📝 Editar proyectos fácilmente
🌐 Mostrar tu trabajo de forma profesional
🔐 Gestionar acceso con autenticación

./mvnw spring-boot:run

# Frontend
cd portfolio-frontend
npm install
npm run dev
