🏗️ Arquitectura
Frontend (React)
      ↓
API REST (Spring Boot)
      ↓
Base de datos (PostgreSQL)
🚀 Ejecutar
# Backend
cd Portifolio-backend
./mvnw spring-boot:run

# Frontend
cd portfolio-frontend
npm install
npm run dev
🌟 Funcionalidades

✔ CRUD de proyectos
✔ Autenticación con JWT
✔ Roles (ADMIN / USER)
✔ Panel admin en dark mode
✔ Preview visual de proyectos
